# Greenland Ice Sheet Analysis

This repository contains code and data for analyzing melt extent of the Greenland ice sheet.

## Datasets

### 2017 Data
- `data/2017_melt_extent.csv`: Daily spatial extent of surface melting for the 2017 season.

## Scripts

- `scripts/process_melt_data.py`: Process melt extent data for analysis.

## Usage

Run the processing script to generate analysis outputs.
